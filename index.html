HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The elements inside the head and body elements are know as child elements.
  • Each element begins with an opening tag and ends with a closing tag.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Don't Repeat Yourself "(DRY)" Code is a coding principal used to reduce the amount of necessary code used to create and style webpages/software.
  • Attributes are variable properties belonging to elements.
  • The class attribute is used to designate specific elements with a unique class so that they can all be selected and styled at once using their unique selector class.
